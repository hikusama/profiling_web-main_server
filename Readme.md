
# Profiling Web 🌐🛜




----------

## Purpose
This system named as SK youth profiling for Baranggay Salaan. It consist 2 develop system mobile app and web based.
The mobile application is used to collect bulk data from the youth of this barrangay. The web based system which is this current repo is used for the main database.

Process:
1. Mobile app collects data from that place.
2. Submit it to web based app.
